# TimingSystem

This repository contains 2 separate projects that are desiged to (potentially) work together:

## 1. SkiTime-RaceReader
- An Arduino project designed to work with the Arduino Uno Wifi Rev2, with the M6E-Nano shield
- sketch that reads tag data, finds the strongest read in a batch, and sends data over wifi soicket

## 2. TrailMeister
- WPF/C# Race software
- Use this software to create a race or training event, and display results while in progress
